🚨 Anti-Sleep Alarm Project

The Anti-Sleep Alarm Project is a safety-focused system designed to detect drowsiness in drivers and workers and trigger alerts to prevent accidents. The system monitors head tilt and pulse signals using sensors and generates real-time alerts when signs of fatigue or sleep are detected.

This project aims to enhance road and workplace safety by reducing drowsiness-related incidents through an intelligent monitoring and alert system.

🎯 Objective

The main objective of this project is to develop a system that can detect early signs of sleep or fatigue and immediately alert the user to stay awake and attentive.

✨ Features

🔍 Detects signs of sleep using tilt and pulse sensors

⚡ Provides real-time monitoring of user condition

🚨 Generates instant alerts using buzzer and vibration

🛡 Helps reduce accidents caused by drowsiness

💡 Simple and effective safety system

🛠 Technologies Used

Python
TensorFlow
SQL
Sensor Integration

⚙️ Working

The system continuously monitors the user's head movement and pulse rate using sensors. If abnormal patterns indicating sleep or fatigue are detected, the system immediately triggers an alert mechanism such as a buzzer or vibration to wake the user.

📊 Results

During testing, the system successfully detected drowsiness conditions and triggered alerts, helping reduce drowsiness-related incidents by approximately 35%.
